Application de gestion de tâches géolocalisées
Ce projet est composé de deux parties :

Frontend : Vue.js (Vite + Pinia)

Backend : AdonisJS (API REST)

Ce README explique comment installer et lancer les deux projets en local
 * Backend — AdonisJS
 - Cloner le projet backend : git clone <url-du-backend> cd backend
 - Installer les dépendances : npm install
 - Configuration des variables d’environnement
 - Générer la clé de l’application : node ace generate:key
 - Lancer les migrations : node ace migration:run
 - Démarrer le serveur backend: npm run dev
* Frontend — Vue.js
 - Cloner le projet frontend : git clone <url-du-backend> cd frontend
 - Installer les dépendances : npm install
 - Démarrer le projet: npm run dev
